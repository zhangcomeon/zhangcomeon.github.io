总结hexo+GitHub搭建博客

#一步安装git和node.js
https://nodejs.org/en/

#第二部打开git bash

安装npm镜像cnpm

安装hexo博客框架，需要借助npm包管理器来安装，国内镜像安装比较慢，先用npm下载cnpm，就是淘宝源，利用淘宝源来安装加快下载速度。

npm install -g cnpm –registry=https://registry.npm.taobao.org

##安装hexo

cnpm install -g hexo-cli

##建立一个博客
{pwd 可以显示所在的文件夹
ls -l 可以显示文件夹中的内容}
1，新建文件夹blog
2，cd blog 进入blog文件夹
3，hexo init 初始化一个博客
4，hexo s 运行本地服务器
5，http://localhost:4000 访问就能看到一个博客文件

##再建立一个博客文件

1.git bash
2.hexo n 文件名
3.cd source/_posts 进入新建的文件夹
4.vim 文件名.cd 进入文件编写博客
5.Esc->:wq 保存并退出vim
6.cd ../.. 返回blog目录下
7.hexo clean
8.hexo g
9.hexo s(现在有个问题，就是hexo s 正常按Ctrl+C就是停止本地服务，但是不好使)

#部署到GitHub上

下载github 插件hexo-deployer-git

cnpm insatll hexo-deployer-git –save
若是显示错误：可以尝试打命令 cnpm install

设置-config.yml文件

1.-config.yml文件在blog目录下
可以用 ls -l 查看blog文件夹
2.vim -config.yml
3.拉到最下面
type：空格git
repo：空格GitHub上仓库的地址
branch: 空格master
4.ESC键->:wq 保存并退出
##部署
hexo d

#登录可查看
